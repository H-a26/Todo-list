Front-End Quest: NSDA Learning Tracker

A progressive todo list tracking my front-end development journey through the NSDA Beginner’s Track. Each task mirrors the curriculum—starting with HTML basics (Month 1), layering CSS (Month 1), then JavaScript interactivity (Month 2-3). Built to evolve as my skills grow, it’s both a practice project and a learning tracker!

  • Month 1 : Web Fundamentals
    • Week 1 : Intro to web development
      • Learn what HTML, CSS, and JavaScript do
      • Understand how websites work
      • Install VS Code + browser dev tools
      • Assignment: Create a personal bio page
    • Week 2 : HTML Basics
      • Study HTML document structure (< !DOCTYPE >,< html >, < head >,< body >)
      • Practice common tags
      • Assignment: create basic ToDo list layout
    • Week 3 : CSS Basics
      • Learn inline vs. internal vs. external CSS
      • Style text (color, font, size) and backgrounds
      • Master the box model (margin, padding, border)
      • Assignment: Style the todo list layout
    • Week 4 : Layout and responsivness
      • Practice CSS Flexbox
      • Try basic CSS Grid
      • Learn media queries for mobile screens
      • Assignment: Make todo mobile friendly
  • Month 2 : JavaScript Fundamentals
    • Week 5 : JS Intro & Setup
      • Link a .js file to HTML
      • Use console.log(), alert(), and prompt()
      • Assignment: use prompt and alert to greet user
    • Week 6 : Variables & Data Types
      • Compare let, const, and var
      • Store tasks as objects/arrays
      • Assignment: create task object with name, completed, id
    • Week 7 : Operators & Conditions
      • Write if/else statements
      • Use logical operators (&&, ||, !)
      • Assignment: show different messages based on task status
    • Week 8 : Loops & Arrays
      • Loop through tasks with for/while
      • Add/remove tasks using .push() and .splice()
      • Assignment: store and display task using arrays and loop
  • Month 3 : Advanced Features
    • Week 9 : Functions & Scope
      • Write reusable functions for adding/deleting tasks
      • Understand global vs. local scope
      • Assignment: create functions add/show task
    • Week 10 : Events & Listeners
      • Add a button to submit tasks
      • Assignment: Add task on button click with confirmition message
    • Week 11 : DOM Manipulation
      • Dynamically update HTML using document.getElementById()
      • Assignment: Add/delete tasks in visible list show dynamic count
    • Week 12 : Deployment
      • combine all features- styling & deployment
      • Assignment: complete Todo App with local storage and deploy it
